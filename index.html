CSS Cheatsheet

Keep track of new CSS concepts here!

    Transform

    Transform objects through rotation, scaling, or skewing using the transform property

    img {  
    rotate(angle);
    transform: scale(x,y);
    transform: skew(angle);
}

    Box Spacing

    Padding adds space within an object, while borders style the direct edges of the padding, and margins add space around the borders

    section {
    padding: 20px; 
    border: 2px;
    margin: 30px;
}

    Header Positioning

    Move header out of box borders.

    .card-header {
    position:relative;
    margin-top: -20px;
}
